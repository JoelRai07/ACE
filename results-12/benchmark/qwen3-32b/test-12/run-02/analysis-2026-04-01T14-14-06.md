### [llm-017] Fehlender Fokus-Trap im Modal
- **WCAG:** 2.4.8 (Level A)
- **Schweregrad:** serious
- **Element/Datei:** styles.css:262
- **Problem:** Das Modal `.modal-box` enthält keine Logik, um den Fokus innerhalb des Modals zu halten.
- **Fix:**
```tsx
// components/Modal.tsx
useEffect(() => {
  if (modalOpen) {
    const firstFocusable = document.querySelector<HTMLElement>('.modal-box [tabindex]');
    const lastFocusable = document.querySelector<HTMLElement>('.modal-box [tabindex]:last-of-type');

    const handleTab = (e: KeyboardEvent) => {
      if (e.key !== 'Tab') return;

      if (!firstFocusable || !lastFocusable) return;

      if (document.activeElement === lastFocusable && !e.shiftKey) {
        firstFocusable.focus();
        e.preventDefault();
      }

      if (document.activeElement === firstFocusable && e.shiftKey) {
        lastFocusable.focus();
        e.preventDefault();
      }
    };

    window.addEventListener('keydown', handleTab);
    firstFocusable?.focus();

    return () => {
      window.removeEventListener('keydown', handleTab);
    };
  }
}, [modalOpen]);
```
