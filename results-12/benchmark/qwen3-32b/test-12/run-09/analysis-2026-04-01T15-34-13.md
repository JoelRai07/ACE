### [llm-006] Modal ohne Fokus-Trap
- **WCAG:** 2.1.2 (Level A)
- **Schweregrad:** critical
- **Element/Datei:** components/ContactForm.tsx:102
- **Problem:** Der modale Dialog hat keinen Fokus-Trap. Der Fokus kann ins Hintergrund-UI wechseln.
- **Fix:**
```tsx
useEffect(() => {
  if (modalOpen) {
    const firstFocusable = document.querySelector<HTMLElement>('#modal-title');
    const lastFocusable = document.querySelector<HTMLElement>('button:last-of-type');

    const handleKeyDown = (e: KeyboardEvent) => {
      if (e.key === 'Tab') {
        if (document.activeElement === lastFocusable && !e.shiftKey) {
          e.preventDefault();
          firstFocusable?.focus();
        } else if (document.activeElement === firstFocusable && e.shiftKey) {
          e.preventDefault();
          lastFocusable?.focus();
        }
      }
    };

    window.addEventListener('keydown', handleKeyDown);
    firstFocusable?.focus();

    return () => {
      window.removeEventListener('keydown', handleKeyDown);
    };
  }
}, [modalOpen]);
```
