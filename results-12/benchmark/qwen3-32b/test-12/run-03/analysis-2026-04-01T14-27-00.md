### [llm-006] Modaler Dialog ohne Fokus-Trap
- **WCAG:** 2.1.2 (Level A)
- **Schweregrad:** critical
- **Element/Datei:** components/ContactForm.tsx:102
- **Problem:** Der modale Dialog hat keinen Fokus-Trap, wodurch der Fokus in den Hintergrund wechseln kann.
- **Fix:**
```tsx
useEffect(() => {
  if (modalOpen) {
    document.body.style.overflow = "hidden";
    const firstFocusable = document.querySelector("#modal-title");
    const lastFocusable = document.querySelector(".btn-primary");
    const trapFocus = (e) => {
      if (e.key !== "Tab") return;
      if (document.activeElement === lastFocusable && !e.shiftKey) {
        firstFocusable.focus();
        e.preventDefault();
      } else if (document.activeElement === firstFocusable && e.shiftKey) {
        lastFocusable.focus();
        e.preventDefault();
      }
    };
    window.addEventListener("keydown", trapFocus);
    return () => {
      window.removeEventListener("keydown", trapFocus);
      document.body.style.overflow = "";
    };
  }
}, [modalOpen]);
```
