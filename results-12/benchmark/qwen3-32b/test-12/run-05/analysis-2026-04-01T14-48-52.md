### [llm-017] Fehlender Fokus-Trap im Modal
- **WCAG:** 2.4.8 (Level AA)
- **Schweregrad:** serious
- **Element/Datei:** styles.css:262-288
- **Problem:** Das Modal `.modal-box` enthält keine Logik für einen Fokus-Trap. Der Fokus kann den Modal verlassen.
- **Fix:**
```tsx
// Beispiel für Fokus-Trap Logik
useEffect(() => {
  if (modalOpen) {
    const firstFocusable = document.querySelector<HTMLElement>('.modal-box [tabindex]');
    const lastFocusable = document.querySelector<HTMLElement>('.modal-box [tabindex]:last-of-type');

    const handleKeyDown = (e: KeyboardEvent) => {
      if (e.key === 'Tab') {
        if (!firstFocusable || !lastFocusable) return;

        if (document.activeElement === lastFocusable && !e.shiftKey) {
          firstFocusable.focus();
          e.preventDefault();
        } else if (document.activeElement === firstFocusable && e.shiftKey) {
          lastFocusable.focus();
          e.preventDefault();
        }
      }
    };

    window.addEventListener('keydown', handleKeyDown);
    firstFocusable?.focus();

    return () => {
      window.removeEventListener('keydown', handleKeyDown);
    };
  }
}, [modalOpen]);
```
