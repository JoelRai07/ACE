### [llm-030] Modaler Dialog ohne Fokus-Trap
- **WCAG:** 2.1.2 (Level A)
- **Schweregrad:** critical
- **Element/Datei:** components/Notifications.tsx:46
- **Problem:** Der modale Dialog hat keine Fokus-Trap-Logik.
- **Fix:**
```tsx
// Fokus-Trap-Logik in useEffect hinzufügen
useEffect(() => {
  const firstFocusable = document.querySelector('[autofocus]');
  const lastFocusable = document.querySelectorAll('button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])')[0];
  const handleKey = (e) => {
    if (e.key === 'Tab') {
      if (document.activeElement === lastFocusable && !e.shiftKey) {
        firstFocusable.focus();
        e.preventDefault();
      } else if (document.activeElement === firstFocusable && e.shiftKey) {
        lastFocusable.focus();
        e.preventDefault();
      }
    }
  };
  window.addEventListener('keydown', handleKey);
  return () => window.removeEventListener('keydown', handleKey);
}, []);
```
