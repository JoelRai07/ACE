### [llm-006] Modal ohne Fokus-Trap
- **WCAG:** 2.1.2 (Level A)
- **Schweregrad:** critical
- **Element/Datei:** components/ContactForm.tsx:102
- **Problem:** Der modale Dialog hat keinen Fokus-Trap, wodurch der Fokus in den Hintergrund gelangen kann.
- **Fix:**
```tsx
// Beispiel für Fokus-Trap mit React
useEffect(() => {
  if (modalOpen) {
    const firstFocusable = document.querySelector('#modal-title');
    const lastFocusable = document.querySelector('#close-button');
    firstFocusable?.focus();
    
    const handleKeyDown = (e: KeyboardEvent) => {
      if (e.key === 'Tab') {
        if (document.activeElement === lastFocusable && !e.shiftKey) {
          e.preventDefault();
          firstFocusable?.focus();
        } else if (document.activeElement === firstFocusable && e.shiftKey) {
          e.preventDefault();
          lastFocusable?.focus();
        }
      }
    };
    
    window.addEventListener('keydown', handleKeyDown);
    return () => window.removeEventListener('keydown', handleKeyDown);
  }
}, [modalOpen]);
```
